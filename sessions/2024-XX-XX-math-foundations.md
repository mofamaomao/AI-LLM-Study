## 核心概念讲解与强化
### 1. 线性代数基础强化
#### 1.1 线性变换：类比理解
> "线性变换就像用放大镜看图片，图片会放大但形状不变"

#### 1.2 关键细节
- **定义**：线性变换是保持向量加法和标量乘法的变换
- **数学表示**：`y = Wx`，其中W是矩阵，x是输入向量，y是输出向量
- **直观理解**：
  - 缩放：向量的长度改变，但方向不变
  - 旋转：向量的方向改变，但长度不变
  - 剪切：向量的形状改变，但保持平行四边形法则

#### 1.3 实际案例
- 在Transformer中，线性变换用于将输入向量转换为Query、Key、Value向量
- 比如输入向量是[1, 2, 3]，矩阵W是[[1, 0, 0], [0, 2, 0], [0, 0, 3]]，输出向量是[1, 4, 9]
- 这个变换将向量的每个维度分别缩放1倍、2倍、3倍

### 2. 自注意力机制数学细节强化
#### 2.1 Query、Key、Value的来源
> "Query、Key、Value是通过对输入向量进行线性变换得到的"

- **输入**：每个词的嵌入向量（比如维度是768）
- **变换矩阵**：Wq、Wk、Wv（都是768×768的矩阵）
- **输出**：
  - Query = 输入向量 × Wq
  - Key = 输入向量 × Wk
  - Value = 输入向量 × Wv
- **为什么需要三个矩阵**：让模型学习到不同的语义表示，分别用于提问、回答和提供内容

#### 2.2 点积的数学含义
> "点积就像两个向量的'相似度分数'，衡量它们的方向是否一致"

- **数学计算**：`点积 = x₁y₁ + x₂y₂ + ... + xₙyₙ`
- **直观理解**：
  - 如果两个向量方向相同，点积为正且最大
  - 如果两个向量方向相反，点积为负且最小
  - 如果两个向量垂直，点积为0
- **在自注意力中的作用**：计算Query和Key的相似度，得到注意力分数

#### 2.3 自注意力完整流程
```
输入向量 → 线性变换 → Query、Key、Value
Query × Key.T → 注意力分数 → 缩放 → Softmax → 注意力权重
注意力权重 × Value → 自注意力输出
```

#### 2.4 缩放因子的作用
- **为什么需要缩放**：当向量维度d_k很大时，点积的结果会很大，导致Softmax输出接近0或1，梯度消失
- **缩放因子**：`√d_k`，其中d_k是Key向量的维度
- **作用**：将注意力分数的方差缩放到1，防止梯度消失
